
仁马小程序接口
    域名      https://lck.hzlyzhenzhi.com
    请求参数    post方式
    返回结构    code (0-报错 1-成功)    message(对应的code信息)  data(返回的数据信息)

    接口名称       组团活动首页
    接口地址        /content/api/group-index
    接口参数        page        页数 默认1 可不传
    响应参数        total           组团活动数量
                    group            组团活动集合

    接口名称       组团活动组团人数
    接口地址        /content/api/group-access
    接口参数        groupId        组团活动id
    响应参数        total           组团人数
                    current         组团用户集合

    接口名称       组团活动商品列表
    接口地址        /content/api/group-list
    接口参数        groupId        组团活动id
                    uid             当前用户id
                    createUid       开团人的uid（参团时进入必传）  开团进入可不传
                    userGroupId     开团人的开团id（参团时进入必传）
    相应参数        shareId         分享id（参团凭证 参团下单时必传）


    接口名称       组团活动商品列表（搜索接口）
    接口地址        /content/api/group-search
    接口参数        groupId        组团活动id
                    search          搜索内容（商品标题）
                    priceMin         最低价
                    priceMax         最高价
                    brand             品牌
                    voltage         电压
                    mileage         续航里程
                    order         价格顺序 1-低到高 2-高到低 0-默认按刷新时间
                    sex             0-通用 1-男 2-女

    接口名称       组团活动商品详情
    接口地址        /content/api/group-detail
    接口参数        groupId          组团活动id
                    productId         商品id
                    uid                 用户id
                    page                页数 默认1 可不传

    相应参数        product下的groupPriceData集合       商品的规范分类数据  catePrice 原本的分类价格 catPriceId  原本的分类id    groupPrice  组团价格    id 组团的分类id  单独购买 传原本的分类参数（catePrice catPriceId） 参团开团传组团分类参数（id groupPrice）


    接口名称       开团/单独购买
    接口地址        /content/api/group-create
    接口参数        groupId        组团活动id
                    uid             用户id
                    productId       商品id
                    type            购买类型 1-单独购买 2-开团
                    catPriceId      分类id 单独购买传原本分类的  开团传组团分类的
                    addressId       地址id
                    extInfo         附加信息

    接口名称       参团
    接口地址        /content/api/group-join
    接口参数        groupId        组团活动id
                    uid             用户id
                    productId       商品id
                    catPriceId      组团分类的id
                    addressId       地址id
                    extInfo         附加信息
                    shareId         开团人记录id（组团活动商品列表接口返回的）

    接口名称        我的组团
    接口地址        /content/api/group-record
    接口参数        uid             用户id
    相应参数        myCreate         我的开团（包含单独购买）  endTime 组团结束时间  status 状态 0-待支付 1-开团成功/参团成功-组团人数不够 2-组团成功 -1 退款成功  join集合 开团下的参团人员
                    myJoin          我的参团  ktrNickname 开团人姓名 ktrAvatar 开团人头像 ktTime  开团时间
